# Penguins Report

## Objective

Find and visualize heuristics for distinguishing between penguin species on Antarctica's Palmer Archipelago

## Dataset

penguins.csv, provided by a researcher at the Palmer Long-Term Ecological Research (LTER) study area, their README and data is available at https://github.com/allisonhorst/palmerpenguins/blob/main/README.md. Otherwise
can be important via Seaborn using sns.load_dataset("penguins").

## Tools

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter

## Findings
* Torgensen island has only Adelie penguins. Biscoe island has Adelie and Gentoo. Dream island has Adelie and Chinstrap.
* Gentoo penguins are larger than the other two species, with bigger bodies and longer flippers.
* Adelie penguins have shorter bill lengths than the other two species.
